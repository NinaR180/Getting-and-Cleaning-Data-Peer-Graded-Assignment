The repository contains data and accompanying documents for the Getting and Cleaning Data final project. 
The author of the repository is Nina Radosic. 

The repository contains the following scripts:
1) CodeBook.txt, which describes the analysis, as well as the data. It is intended to provide an explanation 
of the procedures for transforming the dataset found on 
http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones into a tidy data set. 
2) TidyData.txt contains the tidy data which are the result of the procedures described in the CodeBook.
3) run_analysis.R is the R script that performs the data transformations as per the project instructions.
